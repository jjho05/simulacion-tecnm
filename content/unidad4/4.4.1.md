# 4.4.1 Pruebas paramétricas

Las pruebas paramétricas asumen que los datos siguen una distribución específica (normalmente la distribución normal) y utilizan parámetros poblacionales para realizar las comparaciones.

## Características de las Pruebas Paramétricas

### Ventajas:
- Mayor potencia estadística cuando se cumplen los supuestos
- Resultados más precisos con muestras pequeñas
- Interpretación directa de parámetros (media, varianza)

### Desventajas:
- Requieren que los datos cumplan supuestos de distribución
- Sensibles a valores atípicos
- Pueden dar resultados erróneos si no se cumplen los supuestos

## Prueba t de Student (Comparación de Medias)

Compara la media del modelo de simulación con la media del sistema real.

### Hipótesis:
- $H_0$: $\mu_{modelo} = \mu_{real}$ (el modelo es válido)
- $H_1$: $\mu_{modelo} \neq \mu_{real}$ (el modelo no es válido)

### Estadístico de Prueba:
$$t = \frac{\bar{X}_{modelo} - \bar{X}_{real}}{s_p\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}}$$

donde:
- $\bar{X}_{modelo}$, $\bar{X}_{real}$ son las medias muestrales
- $s_p$ es la desviación estándar combinada
- $n_1$, $n_2$ son los tamaños de muestra

### Desviación Estándar Combinada:
$$s_p = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}}$$

### Decisión:
Si $|t| < t_{\alpha/2, n_1+n_2-2}$, se acepta $H_0$ (el modelo es válido).

### Ejemplo:
**Sistema:** Tiempo de servicio en un banco

**Datos reales:** 30 observaciones, $\bar{X}_{real} = 5.2$ min, $s_{real} = 1.5$ min
**Datos simulados:** 30 réplicas, $\bar{X}_{sim} = 5.5$ min, $s_{sim} = 1.4$ min

**Cálculo:**
1. $s_p = \sqrt{\frac{29(1.5)^2 + 29(1.4)^2}{58}} = 1.45$
2. $t = \frac{5.5 - 5.2}{1.45\sqrt{2/30}} = 0.80$
3. Con $\alpha = 0.05$, $t_{0.025,58} = 2.00$
4. Como $|0.80| < 2.00$, se acepta que el modelo es válido

## Prueba F (Comparación de Varianzas)

Compara la variabilidad del modelo con la del sistema real.

### Hipótesis:
- $H_0$: $\sigma_1^2 = \sigma_2^2$ (las varianzas son iguales)
- $H_1$: $\sigma_1^2 \neq \sigma_2^2$ (las varianzas son diferentes)

### Estadístico de Prueba:
$$F = \frac{s_1^2}{s_2^2}$$

donde $s_1^2 > s_2^2$ (siempre colocar la mayor varianza en el numerador)

### Decisión:
Si $F < F_{\alpha, n_1-1, n_2-1}$, se acepta $H_0$ (las varianzas son iguales).

### Ejemplo:
Usando los mismos datos del ejemplo anterior:

**Cálculo:**
1. $F = \frac{1.5^2}{1.4^2} = \frac{2.25}{1.96} = 1.15$
2. Con $\alpha = 0.05$, $F_{0.05,29,29} = 1.86$
3. Como $1.15 < 1.86$, se acepta que las varianzas son iguales

## Prueba Z (Para Muestras Grandes)

Cuando $n > 30$, se puede usar la distribución normal estándar.

### Estadístico:
$$Z = \frac{\bar{X}_{modelo} - \bar{X}_{real}}{\sqrt{\frac{s_1^2}{n_1}+\frac{s_2^2}{n_2}}}$$

### Decisión:
Si $|Z| < Z_{\alpha/2}$, se acepta $H_0$.

## Análisis de Varianza (ANOVA)

Para comparar múltiples configuraciones del modelo simultáneamente.

### Hipótesis:
- $H_0$: $\mu_1 = \mu_2 = ... = \mu_k$ (todas las medias son iguales)
- $H_1$: Al menos una media es diferente

### Estadístico F:
$$F = \frac{MSB}{MSW}$$

donde:
- $MSB$ = Media cuadrática entre grupos
- $MSW$ = Media cuadrática dentro de grupos

## Código de Ejemplo en Python

```python
import numpy as np
from scipy import stats

def prueba_t_validacion(datos_reales, datos_simulados, alpha=0.05):
    """
    Realiza prueba t para validar modelo de simulación
    """
    # Calcular estadístico t
    t_stat, p_value = stats.ttest_ind(datos_reales, datos_simulados)
    
    # Decisión
    if p_value > alpha:
        resultado = "VÁLIDO"
        mensaje = f"No se rechaza H0 (p={p_value:.4f})"
    else:
        resultado = "NO VÁLIDO"
        mensaje = f"Se rechaza H0 (p={p_value:.4f})"
    
    return {
        'estadistico_t': t_stat,
        'p_value': p_value,
        'resultado': resultado,
        'mensaje': mensaje
    }

def prueba_f_varianzas(datos_reales, datos_simulados, alpha=0.05):
    """
    Realiza prueba F para comparar varianzas
    """
    var1 = np.var(datos_reales, ddof=1)
    var2 = np.var(datos_simulados, ddof=1)
    
    # F siempre con la mayor varianza en numerador
    if var1 > var2:
        F = var1 / var2
        df1, df2 = len(datos_reales)-1, len(datos_simulados)-1
    else:
        F = var2 / var1
        df1, df2 = len(datos_simulados)-1, len(datos_reales)-1
    
    # Valor crítico
    F_critico = stats.f.ppf(1-alpha, df1, df2)
    
    return {
        'estadistico_F': F,
        'F_critico': F_critico,
        'resultado': 'Varianzas iguales' if F < F_critico else 'Varianzas diferentes'
    }

# Ejemplo de uso
datos_reales = np.array([5.1, 4.9, 5.3, 5.0, 5.2, 4.8, 5.4, 5.1])
datos_simulados = np.array([5.3, 5.0, 5.5, 5.2, 5.1, 5.4, 5.0, 5.3])

resultado_t = prueba_t_validacion(datos_reales, datos_simulados)
print(f"Prueba t: {resultado_t['resultado']}")
print(f"  {resultado_t['mensaje']}")

resultado_f = prueba_f_varianzas(datos_reales, datos_simulados)
print(f"\nPrueba F: {resultado_f['resultado']}")
print(f"  F = {resultado_f['estadistico_F']:.4f}")
```
