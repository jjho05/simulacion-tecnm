# Unidad 4: Lenguajes de Simulación

## Descripción

Esta unidad cubre las herramientas de simulación (lenguajes y software), su aplicación a casos reales, y el análisis estadístico de resultados.

## Contenido

### [4.1 Lenguaje de simulación y simuladores](4.1.md)
- Lenguajes de propósito general (Python/SimPy, MATLAB)
- Lenguajes especializados (GPSS, SIMSCRIPT)
- Simuladores comerciales (Arena, ProModel, FlexSim, AnyLogic)
- Tabla comparativa y criterios de selección

### [4.2 Aprendizaje y uso de un lenguaje de simulación o simulador](4.2.md)
- Tutorial completo de SimPy
- Guía práctica de Arena
- Conceptos: Environment, Process, Resource
- Ejemplos completos M/M/1 y M/M/c
- Mejores prácticas de modelado

### [4.3 Aplicación de un lenguaje o simulador a un caso específico](4.3.md) *(Archivo padre)*
- Metodología de aplicación (7 pasos)
- Recolección y análisis de datos
- Verificación y validación
- Experimentación

#### [4.3.1 Sistemas de líneas de espera](4.3.1.md)
- Notación de Kendall (A/B/c/K/N/D)
- Sistema M/M/1 (teoría y simulación)
- Sistema M/M/c (teoría y simulación)
- Casos de estudio: Hospital, Banco
- Análisis económico

#### [4.3.2 Sistemas de inventarios](4.3.2.md)
- Políticas (Q,R), (s,S), (T,S)
- Modelo EOQ (Economic Order Quantity)
- Simulación de inventarios estocásticos
- Casos de estudio: Supermercado, Manufactura

### [4.4 Análisis de resultados de la simulación](4.4.md) *(Archivo padre)*
- Importancia de la validación
- Metodología de validación (5 pasos)
- Análisis de sensibilidad
- Mejores prácticas

#### [4.4.1 Pruebas paramétricas](4.4.1.md)
- Prueba t de Student (comparación de medias)
- Prueba F (comparación de varianzas)
- Prueba Z (muestras grandes)
- ANOVA (múltiples configuraciones)
- Implementación en Python

#### [4.4.2 Pruebas no paramétricas](4.4.2.md)
- Prueba de Kolmogorov-Smirnov
- Prueba de Mann-Whitney
- Prueba de los signos
- Prueba de rachas
- Comparación con pruebas paramétricas

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Seleccionar la herramienta de simulación apropiada
- ✅ Implementar modelos en SimPy y Arena
- ✅ Modelar sistemas de colas e inventarios
- ✅ Validar modelos usando pruebas estadísticas
- ✅ Analizar y comparar escenarios

## Recursos Adicionales

- **Software:** SimPy, Arena, Python (SciPy)
- **Libros:** Banks et al. (2010), Law (2015), Gross & Harris (2008)
- **Tiempo estimado:** 14-16 horas
