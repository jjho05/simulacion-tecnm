# Unidad 5: Proyecto Integrador

## Descripción

Esta unidad final integra todos los conocimientos del curso en un proyecto completo de simulación de un sistema real. Es la aplicación práctica de la metodología completa de simulación.

## Contenido

### [5.1 Análisis y modelado de un sistema para su estudio mediante simulación](5.1.md)

**Metodología de 7 Fases:**

1. **Definición del Problema**
   - Identificar sistema a estudiar
   - Definir objetivos específicos
   - Establecer alcance

2. **Recolección de Datos**
   - Tiempos de proceso
   - Tiempos entre llegadas
   - Recursos disponibles
   - Análisis estadístico

3. **Construcción del Modelo**
   - Seleccionar herramienta
   - Desarrollar modelo conceptual
   - Implementar en código

4. **Verificación**
   - Debugging del código
   - Pruebas de lógica
   - Trazas de eventos

5. **Validación**
   - Comparación con sistema real
   - Pruebas estadísticas (t-test, K-S)
   - Validación con expertos

6. **Experimentación**
   - Diseño de escenarios
   - Ejecución de réplicas
   - Análisis comparativo

7. **Documentación y Presentación**
   - Reporte técnico
   - Visualizaciones
   - Recomendaciones

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Desarrollar un proyecto completo de simulación
- ✅ Aplicar la metodología de 7 fases
- ✅ Recolectar y analizar datos reales
- ✅ Validar modelos rigurosamente
- ✅ Proponer mejoras fundamentadas
- ✅ Comunicar resultados profesionalmente

## Ejemplos de Sistemas Apropiados

**Manufactura:**
- Línea de ensamble
- Sistema de pintura
- Proceso de empaque

**Servicios:**
- Banco (cajas)
- Hospital (emergencias)
- Call center

**Logística:**
- Centro de distribución
- Almacén
- Sistema de transporte

**Retail:**
- Supermercado
- Tienda departamental
- E-commerce fulfillment

## Entregables

1. **Código fuente** del modelo
2. **Reporte técnico** completo
3. **Presentación** (15-20 min)
4. **Datos** utilizados

## Recursos Adicionales

- **Templates:** Formato de reporte, checklist de verificación
- **Software:** SimPy, Arena, Python
- **Tiempo estimado:** 20-30 horas
